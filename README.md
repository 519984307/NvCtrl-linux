## Installation
### Arch GNU/Linux
Download PKGBUILD file from latest release

### Building from sources
```
$ git clone --recurse-submodules https://codeberg.org/AlexCr4ckPentest/GWEpp
$ cd GWEpp
$ mkdir build && cd build
$ cmake ..
$ make -j$(nproc)
```
